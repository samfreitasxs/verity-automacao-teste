# Avaliacção Tecnica

## Technologies

- Node.js
- JavaScript
- Cypress

## Configurando o ambiente

1. Instale o Node.js.

2. É necessário instalar dependências, digite no prompt de comando:
```
npm i
```
3. Para abrir a execução dos testes utilize:
```
npx cypress open

Para executar no modo headless:

npx cypress run
```
